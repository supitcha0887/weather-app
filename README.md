# Weather App 🌤️

## 📌 About This Project

This project is a simple **weather application** that allows users to check real-time weather information for different locations.

The application retrieves weather data from an external API and displays it in a user-friendly interface.

---

## 🎯 Objectives

* Practice working with APIs
* Learn how to fetch and display real-time data
* Improve frontend development skills

---

## 🚀 Features

* Search weather by city name
* Display current temperature
* Show weather conditions (e.g. cloudy, rainy)
* Display additional information such as humidity and wind speed
* Simple and responsive UI

---

## 🛠 Tech Stack

* **Frontend:** HTML, CSS, JavaScript *(or React if you used it)*
* **API:** Weather API (e.g. OpenWeatherMap)

---

## ⚙️ How It Works

* User enters a city name
* The app sends a request to the weather API
* The API returns weather data
* The app displays the data on the screen

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/supitcha0887/weather-app
   ```
2. Open the project folder
3. Run the project (e.g. open `index.html` or start with npm)
